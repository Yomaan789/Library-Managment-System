# Library Management System (Core Java)

## Overview
A console-based Library Management System built using Core Java.

## Features
- Add Users (Members)
- Add Books
- Issue Books
- Return Books
- View Available Books

## Concepts Used
- Object-Oriented Programming (Abstraction, Inheritance, Polymorphism)
- Java Collections (HashMap, ArrayList)
- Interface Implementation
- Encapsulation
- Console-based interaction

## Design Decisions
- Centralized storage inside Library class
- Abstract User class extended by Member
- Book availability tracked inside system

## How to Run
Compile and run Main.java
